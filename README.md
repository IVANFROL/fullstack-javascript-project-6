## Fullstack JavaScript Project 6

Проект на Fastify для Hexlet.

### Установка

```bash
npm install
```

### Запуск

```bash
npm start
```

Для разработки:
```bash
npm run dev
```

### Переменные окружения

Скопируйте `.env.example` в `.env` и настройте необходимые переменные:

```bash
cp .env.example .env
```
